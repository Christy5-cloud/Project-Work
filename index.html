The Basic Language of Web: HTML

Posted by Laura Jones on Monday, October 14th, 2024.

All modern websites and web applications are built using three fundamantal technologies: HTML, CSS and JavaScript. These are the languages of the web

In this post, let's focus on HTML. We will learn what HTML is all about and why you too should learn it

What is HTML?

HTML stands for Hyper Text Markup Language that web developer use to structure and describe the content of a web page

HTML consists of elements that describes types of content: paragraghs, links, headings, images, videos,e.t.c. web browsers understand HTML

In HTML, each element is made up of 3 parts:

  • The opening tag
  • The closing tag
  • The actual element
